## 🚀 Motivação
Este é um projeto para entender melhor os conceitos de Laravel na versão 8.0, e também treinar programação backend e frontend.


### Ajustes e melhorias
O projeto ainda está em desenvolvimento e as próximas atualizações serão voltadas nas seguintes tarefas:

- [x] Criação de um sistema básico de login
- [] Criação de um sistema básico de avatares customizados
- [] Criação de quadros para armazenamento de listas
- [] Criação de listas para armazenamento de todos
- [] Colaboração nos projetos
